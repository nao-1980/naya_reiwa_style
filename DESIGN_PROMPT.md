## デザイン要件

### 2026年トレンド要素（必須）

1. **グラスモーフィズム**
   - 半透明の背景（rgba）
   - backdrop-filter: blur(20px)効果
   - 繊細なボーダーと影

2. **3D効果とアニメーション**
   - 浮遊するグラデーション球体（パララックス効果）
   - ホバー時の3D変形（transform: translateY, scale, rotateY）
   - スムーズなトランジション（cubic-bezier）

3. **大胆なタイポグラフィ**
   - Google Fonts: Poppins（見出し用）、Inter（本文用）
   - 大きな見出しサイズ（clamp関数でレスポンシブ）
   - グラデーションテキスト効果

4. **カラーグラデーション**
   - Gradient 1: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
   - Gradient 2: linear-gradient(135deg, #f093fb 0%, #f5576c 100%)
   - Gradient 3: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%)

5. **マイクロインタラクション**
   - リップル効果（ボタンクリック時）
   - ホバーアニメーション
   - スクロールトリガーアニメーション
   - フローティングラベル（フォーム）

6. **ダークモード対応**
   - ライト/ダークテーマ切り替え
   - LocalStorageで設定保存
   - CSS変数でテーマ管理

## アニメーション仕様

### ヒーロータイトル:
```css
@keyframes slideUp {
    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}
```
- 各行を0.2秒ずつ遅延して表示

### フローティングカード:
```css
@keyframes cardFloat {
    0%, 100% { transform: translateY(0px); }
    50% { transform: translateY(-20px); }
}
```
- duration: 6秒
- 各カードに0.2秒の遅延

### グラデーション球体:
```css
@keyframes float {
    0%, 100% { transform: translate(0, 0) scale(1); }
    33% { transform: translate(30px, -30px) scale(1.1); }
    66% { transform: translate(-20px, 20px) scale(0.9); }
}
```
- duration: 20秒
- 各球体に5秒の遅延
- filter: blur(80px)
