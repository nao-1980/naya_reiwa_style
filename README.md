# NAYA WORKS - 2026 Trend UI Website

## プロジェクト概要

2026年のWebデザイントレンドを取り入れた、個人事業主向けのモダンなポートフォリオWebサイトです。ビジュアル&デザイン、コミュニケーション、UI/UX相談&AI構築の3つの主要サービスを展開するプロフェッショナルなサイトです。

## 🎨 デザイン特徴

### 2026年トレンド要素
- **グラスモーフィズム**: 半透明の背景とブラー効果を使用したモダンなカードデザイン
- **3D効果とアニメーション**: 浮遊するグラデーション球体と滑らかなトランジション
- **大胆なタイポグラフィ**: Poppins & Interフォントを使用した力強いテキスト表現
- **グラデーション**: 美しいカラーグラデーションを多用した視覚的インパクト
- **マイクロインタラクション**: ホバー、クリック、スクロールに反応する細かなアニメーション
- **ダークモード対応**: ライト/ダークテーマの完全サポート

### ビジュアル要素
- **アニメーション効果**: スムーズなページトランジションとスクロールアニメーション
- **レスポンシブデザイン**: デスクトップ、タブレット、モバイル全てに最適化
- **インタラクティブUI**: リップル効果、パララックススクロール、動的カーソル効果

## 🔧 カスタマイズ方法

### カラーの変更
`css/style.css`の`:root`セクションでCSS変数を編集してください。

```css
:root {
    --primary: #6366f1;
    --secondary: #8b5cf6;
    --accent: #ec4899;
    /* 他の変数 */
}
```

### コンテンツの変更
`index.html`内の各セクションを編集してください。
- ヒーローセクションのテキスト
- サービス内容
- ポートフォリオアイテム
- コンタクト情報

### アニメーションの調整
`css/style.css`内の`@keyframes`と`animation`プロパティを編集してください。

## 🌟 主な機能の使い方

### ダークモード切り替え
右上の月/太陽アイコンをクリックすると、ライト/ダークモードが切り替わります。設定はローカルストレージに保存されます。

### フォーム送信
お問い合わせフォームは現在デモモードです。実際の送信機能を実装する場合は、`js/main.js`の`initFormHandling()`関数内のAPI呼び出しコードを追加してください。

```javascript
// 例: 実際のAPI呼び出し
const response = await fetch('/api/contact', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify(formData)
});
```
